### Current Development Focus

Ship and harden the **equity market map and portfolio analytics** platform: single-universe screener with a top-bar **Manage Tickers** modal (CSV Import / Paste Tickers / Current) for ingestion; Yahoo HTTP price/benchmark ingest to Postgres; market-map matrix API + heatmap UI rendered as an **expandable Sector → Sub-Theme → Ticker tree** (no row-level filter; per-row chevrons + Expand all / Collapse all toggle); portfolio weights + benchmark analytics; keep financial math in `src/domain` and services out of presentational components.

### Key Decisions / Constraints

- **Single universe:** App operates as a single "universe of stocks" screener. `GET /api/universe/default` returns or lazily creates the one active universe (DB schema remains multi-universe-capable; UI only exposes one).
- **Hierarchy (only):** Sector → Sub-Theme → Company. Drives market map rows, drill-down, rankings, and aggregations (parent metric = **average of leaf company metrics** unless later specified otherwise).
- **Trading-day semantics:** 1/5/21/63/126/252 trading days for 1D, 5D, 1M, 3M, 6M, 1Y; not calendar days.
- **Returns:** From **adjusted** close; daily return = \(P_t/P_{t-1} - 1\). Volatility: annualized **realized** vol from daily returns, \(\sigma_{ann} = \text{stdev} \times \sqrt{252}\). Sharpe uses configurable risk-free; zero-vol edge case must not divide by zero.
- **Stack:** Next.js + TypeScript, Prisma + PostgreSQL, market data **behind interfaces**; default adapter uses Yahoo **public** chart + quote HTTP (no API key; unofficial—see `docs/ARCHITECTURE.md`). Benchmarks: S&P 500, NASDAQ, DOW stored in `BenchmarkPriceHistory`.
- **Quality:** No analytics logic in React components—call services/API from UI.

- **2026-04-18 (eve, +5):** Fixed silent Apply failures on large pastes (~700+ tickers). `replaceUniverseConstituents` was running 1 `upsert` + 1 `create` **per row** inside a single Prisma `$transaction`, blowing the default 5 s interactive-transaction timeout long before the 743-row paste finished.
